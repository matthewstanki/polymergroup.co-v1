## User Stories

- WordPress Engine.
- Fully working and responsive.
- Desktop, mobile and tablet compatible.
- Version 1.
- 5 languages.
- CTA working.
- SSL done.
- 10 pages on the website.
- Ready for optimalization and SEO.
